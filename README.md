# 键途 KeyPath — 打字成长之路

面向中职/高中学生（15-18 岁）的打字练习平台。Vue 3 + Vite + Naive UI + Pinia + Supabase + ECharts。

## 🚀 快速开始

### 1. 初始化数据库（只需一次）

打开 [Supabase 控制台](https://supabase.com/dashboard) → 你的项目 → **SQL Editor**，
将 [`supabase/schema.sql`](supabase/schema.sql) 的全部内容粘贴进去并点击 **Run**。

完成后会自动创建：

- 全部 11 张数据表 + 行级安全策略
- 登录/导入等安全函数
- 默认超级管理员：账号 **admin**，密码 **admin123**（请登录后立即修改）
- 17 个预设成就 + 10 篇内置文稿

### 2. 本地运行

```bash
npm install
npm run dev
```

浏览器打开 http://localhost:5180

### 3. 部署（Cloudflare Pages）

```bash
npm run build
```

将项目推到 GitHub，在 Cloudflare Pages 中关联仓库：
构建命令 `npm run build`，输出目录 `dist`。

## 👥 使用流程

1. **超级管理员**（admin）登录 → 教学管理 → 教师账号：创建各班教师
2. **教师**登录 → 学生管理：粘贴「学号,姓名,班级」批量导入学生（班级自动创建，默认密码 123）
3. **教师**发布任务：选文稿、设起止时间/时长/重复规则、勾选班级与学生
4. **学生**用「学号 + 姓名 + 密码」登录，首次登录引导选头像、改密码
5. 学生练习/游戏/做任务 → 自动记录成绩、打卡、解锁成就 → 排行榜实时更新

## ✨ 功能总览

- **打字练习**：10 种键位练习（基准键位/全字母/数字/符号/大小写/左右手/弱指）、内置文稿库（中/英/数字符号 × 三档难度）、自定义文稿（粘贴或上传 .txt/.docx）、短文自动循环、CPM/WPM/准确率/错误数/双时长统计
- **6 款游戏**：🚀 太空射击、🌊 文字消消乐、🏃 赛跑竞速（支持房间码多人实时对战，基于 Supabase Realtime）、💣 拆弹游戏、🐍 文字冒险（分支结局）、🎯 打地鼠
- **任务系统**：班级/个别学生指派、起止时间、限时练习、重复提交规则（取最高/最后）、草稿/进行中/截止/归档
- **数据中心**：成绩趋势双轴折线图、错误键位热力图 + 薄弱键位推荐、任务成绩列表、CSV 导出（教师）
- **排行榜**：全体/班级/游戏/进步之星，周榜/月榜/总榜，按速度/准确率/综合
- **成就徽章**：17 个成就（含 3 个隐藏彩蛋），阈值支持教师在后台自定义
- **打卡**：GitHub 风格热力日历、连续打卡统计、班级出勤总览
- **个性化**：10 套预设主题 + 自定义配色、纯色/渐变/图片背景（可调模糊）、DiceBear 头像、4 种键盘音效、字体/行距调节

## 🔑 账号说明

| 角色 | 登录方式 | 默认 |
|------|---------|------|
| 超级管理员 | 账号 + 密码 | admin / admin123 |
| 教师 | 账号 + 密码 | 由超管创建 |
| 学生 | 学号 + 姓名 + 密码 | 密码 123 |
| 访客 | 免登录 | 可练习/玩游戏，不保存成绩 |

> 注：本应用采用面向校园场景的轻量自定义认证（密码以 bcrypt 散列存储、登录走数据库安全函数、密码列对客户端不可读）。适合班级内网/教学使用，不建议存放敏感信息。
